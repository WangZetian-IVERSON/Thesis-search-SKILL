# Citation Style

## APA 7

Use APA 7 for English-language reports unless the user requests another style.

## BibTeX

Generate BibTeX for reference managers and reproducibility.

## GB/T 7714

Use GB/T 7714 for Chinese thesis workflows when requested. Future versions should add deterministic formatting for journal articles, conference papers, books, and dissertations.

## Rule

Never fabricate missing citation fields. Leave absent fields blank or mark them as unavailable.
