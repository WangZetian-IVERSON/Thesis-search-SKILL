# Report Template

## Required Sections

1. Cover: topic, date, databases, number of papers, version.
2. Executive summary: strongest findings and reading recommendations.
3. Search method: query terms, sources, filters, inclusion/exclusion rules.
4. Paper overview table: title, authors, year, venue, DOI/URL, full-text status.
5. Per-paper reading: use the paper reading template.
6. Figure/table evidence: screenshots and manifest details.
7. Cross-paper synthesis: themes, methods, findings, disagreements, gaps.
8. Citations: APA, BibTeX, or GB/T 7714.
9. Audit appendix: source verification, figure/table checks, unresolved risks.

## HTML First

Generate HTML before PDF. HTML is easier to inspect, link, and audit. Export PDF only after images and links are checked.
