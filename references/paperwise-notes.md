# Paperwise Reference Notes

Paperwise is useful as a product reference for readable HTML annotation.

## Adopt

- Command-style workflow: find, read, digest, cite.
- Dual-column HTML: original text on the left, annotation cards on the right.
- Color-coded logical roles.
- Question-driven reading mode.
- Quality bar with source, venue, year, and citation status.
- Self-contained HTML output.

## Extend

This project adds stronger provenance and audit layers:

- Multi-source metadata verification.
- Figure/table screenshot manifest.
- Distinction between metadata-only and full-text reading.
- Cross-paper synthesis.
- Final deterministic audit before delivery.
